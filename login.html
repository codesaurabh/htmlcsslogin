Instructions for filling the login form

The user name field is required and has to be compliant with the following rules:
  • The length of the user name must be between 6 and 15 characters.
  • The username must contain following types of characters
    1. English uppercase letters [A-Z]
    2. English lowercase letters [a-z]
    3. Arabic numerals [0-9]
The password field is required and has to be compliant with the following rules:
  • The length of the password must be between 8 and 12 characters.
  • The password must contain following types of characters
    1. Atleast one english uppercase letter [A-Z]
    2. Atleast one english lowercase letter [a-z]
    3. Atleast one arabic numeral [0-9]
